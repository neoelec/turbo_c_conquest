# 제8장: 수학 함수와 그래픽 함수

## 1. 수학 함수 (Mathematical Functions)
수학 함수를 사용할 때는 해당 함수가 선언된 헤더 화일인 `<math.h>` 또는 `<stdlib.h>`를 반드시 소스 코드 선두에 포함시켜야 합니다.

### 1.1 주요 수학 함수의 종류
- **삼각함수**: `sin`, `cos`, `tan`, `asin`, `acos`, `atan` 등. (매개변수는 호도(radian) 단위를 사용함)
- **지수 및 대수함수**: `exp`(${e}^{x}$), `log`($ln\left(x\right)$), `log10`(${log}_{10}\left(x\right)$), `pow`(${x}^{y}$), `sqrt`($\sqrt{x}$) 등.
- **절대값 및 수치 관련**: `abs` (정수), `fabs` (실수), `max` (최대값), `min` (최소값) 등.
- **난수 함수**: `randomize` (난수 발생기 초기화), `random` (난수 생성).

### 1.2 주요 매크로 상수 (`math.h`)
정밀한 계산을 위해 미리 정의된 상수들을 활용할 수 있습니다.
- `M_PI`: 원주율 ($\pi$)
- `M_E`: 자연로그의 밑 ($e$)
- `M_SQRT2`: $\sqrt{2}$

## 2. 그래픽 함수의 기초
터보 C는 90여 개의 방대한 그래픽 라이브러리를 제공하며, 이를 통해 도형 그리기와 색상 제어가 가능합니다.

### 2.1 그래픽 시스템 초기화 및 종료
그래픽 작업을 시작하고 마칠 때는 전형적인 절차를 따라야 합니다.

1. **초기화**: `initgraph` 함수를 사용해 그래픽 어댑터와 모드를 설정합니다.
2. **본체 실행**: 원하는 그래픽 함수들로 화면을 구성합니다.
3. **종료**: `closegraph` 함수를 호출하여 그래픽 시스템을 종료하고 텍스트 모드로 복귀합니다.

### 2.2 그래픽 화면 좌표계
- **절대 좌표계**: 화면의 좌상단을 원점 (0, 0)으로 합니다.
- **뷰포트(Viewport)**: 화면의 일부 영역을 설정하여 별도의 상대 좌표계를 사용할 수 있습니다.

## 3. 주요 그래픽 함수 및 활용

### 3.1 도형 그리기 (Drawing Functions)
- **선(Line)**: `line` (두 점 연결), `lineto` (현재 위치에서 특정 점까지), `linerel` (상대적 거리만큼 그리기).
- **원(Circle)**: `circle(x, y, radius)`. 화면의 **종횡비(Aspect Ratio)** 에 따라 좌표 보정이 필요할 수 있습니다.
- **기타**: `rectangle` (직사각형), `ellipse` (타원).

### 3.2 채색 및 화면 조작
- **채색**: `floodfill` 함수로 폐곡선의 내부를 특정 무늬나 색상으로 채웁니다. `setfillstyle`을 통해 무늬를 선택할 수 있습니다.
- **화면 소거**: `cleardevice` 함수로 화면 전체를 지웁니다.

### 3.3 그래픽 모드에서의 텍스트 출력
그래픽 모드에서는 표준 `printf`를 직접 사용할 수 없으므로 다음의 과정을 권장합니다.

1. 수치 데이터를 `itoa` 또는 `sprintf` 함수를 사용하여 **문자열로 변환**합니다.
2. `outtext` 또는 `outtextxy` 함수를 사용해 화면에 출력합니다.
